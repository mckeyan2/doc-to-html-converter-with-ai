# doc→html

Convert Word, Excel & typed text to clean, semantic HTML5 — entirely in your browser.

**Live app:** https://mckeyan2.github.io/doc-to-html-converter-with-ai/

## Features

- Upload `.docx`, `.xlsx`, `.xls`, or `.csv` files — or type directly in the built-in rich-text editor
- Runs 100% client-side; no files are uploaded to any server
- Outputs clean, semantic HTML5 with no inline styles or class attributes
- Built-in conversion test suite verifies all transform rules

## Supported formats

| Format | Notes |
|--------|-------|
| `.docx` | Word documents via Mammoth.js |
| `.xlsx` / `.xls` | Excel spreadsheets via SheetJS |
| `.csv` | CSV files via SheetJS |
| Text editor | Rich-text editor with formatting toolbar |
