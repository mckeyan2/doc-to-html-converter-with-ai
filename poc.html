How It Works
Four simple steps — files never leave your device. Claude API key is optional and only used for unrecognised rules.
1

Choose Input

Upload a .docx / .xlsx / .csv file via drag-and-drop, or switch to the built-in rich text editor.

2

Add Smart Rules ✨ Claude AI

Type plain-English instructions in the Smart Rule Builder. Known patterns apply instantly in-browser; add a Claude API key to handle any instruction Claude understands.

3

Convert

15 built-in rules run first (toggle to disable them), then your custom rules apply on top. Or use only your rules for full control.

4

Copy, Download & Verify

Preview and copy clean HTML. 22 automated checks validate every conversion — download a full test report.
